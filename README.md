## Ejecución

Para ejecutar el agente:

python run_2048.py --mode agent --agent-module submission --agent-class Agent --episodes 50
Nota sobre el tiempo de ejecución

El agente está optimizado para cumplir el límite de 1 minuto en Colab/CPU según lo estipulado.
Sin embargo, debido a variaciones normales del entorno de ejecución (carga de CPU, recursos compartidos, etc.), en algunas ocasiones el tiempo total puede subir ligeramente a aproximadamente 1 minuto con 7–10 segundos.

Agradecemos la comprensión respecto a esta pequeña variación, que no depende directamente de la lógica del agente sino del rendimiento momentáneo del entorno.
